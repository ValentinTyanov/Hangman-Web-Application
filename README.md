# HangmanWebApplication
Java Web Application based on the well-known Hangman game. It uses: Spring Framework - Spring Boot, Spring Data & Spring Web, Apache Shiro for FORM Authentication, ReactJS & some JSPs as FrontEnd, RESTful Web Services, MySQL, JPA Metamodel & Hibernate for ORM.
Unit tests are performed using JUnit 5, AssertJ and Mockito. Integration test suite is implemented with Selenium Webdriver.
